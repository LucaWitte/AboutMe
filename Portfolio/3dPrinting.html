Ender 3

I am using an Ender 3 with the silent board and glass bed modifications. Some technical details will be added later.

As I used blender for 3d modeling before, I have tried to create to-scale models in Blender. This works surprisingly well if the 3Dprint tools extension is used to prevent disconnected vertices. There is a ton of measuring options to scale things correctly. I have worked with CAD software in the past when designing microfluidic chips, but Blender is definitely more visually appealing and provides more options if looks of the model are more important than optimizing strength or similar.
